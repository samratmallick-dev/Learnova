## 📖 About

Learnova is a full-featured Learning Management System designed to facilitate online education. It provides a seamless experience for students to learn, instructors to teach, and administrators to manage the entire platform efficiently.

## ✨ Features

### 👨‍🎓 Student Features

- **Authentication & Profile**
  - User registration and secure login system
  - Profile management and password update

- **Course Management**
  - Browse and search available courses
  - Course enrollment and purchase (if paid courses are enabled)
  - Add courses to wishlist or favorites

- **Learning Experience**
  - Access video lectures, notes, and learning materials
  - Track learning progress and completed lessons
  - Download study materials (PDF, documents, etc.)
  - Discussion or comment section for course interaction

- **Assessment & Certification**
  - Take quizzes and assessments
  - View scores and performance reports
  - Certificate generation after course completion

---

### 👨‍🏫 Instructor Features

- **Profile & Course Creation**
  - Instructor registration and profile management
  - Create, edit, and publish courses
  - Organize course content into modules or sections

- **Content Management**
  - Upload video lectures, documents, and other learning resources
  - Create quizzes, assignments, and assessments
  - Manage course updates and announcements

- **Student Engagement**
  - Monitor enrolled students
  - Track student progress and performance analytics
  - Respond to student questions and discussions

- **Analytics & Revenue**
  - View course ratings and feedback from students
  - Revenue tracking (if course selling feature is included)

---

### 👨‍💼 Admin Features

- **User Management**
  - Secure admin dashboard access
  - Manage all users (students and instructors)
  - Approve or reject instructor registrations

- **Content Moderation**
  - Approve, edit, or remove courses
  - Manage categories and course listings
  - Handle reported content or user issues

- **Platform Analytics**
  - Monitor platform activity and analytics
  - View platform statistics (total users, courses, enrollments)
  - Manage payments and transactions (if applicable)

- **System Administration**
  - Send platform-wide notifications or announcements
  - Maintain system security and data management

---

## 🛠️ Tech Stack


### Frontend
- **React.js** - UI library for building interactive interfaces
- **Tailwind CSS v4** - Utility-first CSS framework
- **Shadcn/UI** - Re-usable component library

### Backend
- **Node.js** - JavaScript runtime environment
- **Express.js** - Web application framework

### Database
- **MongoDB** - NoSQL database for flexible data storage
